🚀 Quick Installation Check

Before testing, ensure:

  1. Extension is loaded in Chrome (chrome://extensions/)
  2. Extension is pinned to toolbar
  3. Terms Analyzer API is running on port 3000
  4. Icons are generated and saved in extension/icons/

🔴 High Risk Website Tests

These sites should show RED "HIGH" badges and warning notifications:

📘 Facebook Terms 🐦 Twitter Terms 📸 Instagram Terms

Expected Results:

  1. Extension badge turns RED with "HIGH" text
  2. Warning notification appears in top-right
  3. Popup shows 70-90 risk score
  4. Multiple concerns listed

🟡 Medium Risk Website Tests

These sites should show YELLOW "MED" badges with moderate warnings:

📦 Amazon Terms 🔍 Google Terms 💻 Microsoft Terms

Expected Results:

  1. Extension badge turns YELLOW with "MED" text
  2. Moderate notification may appear
  3. Popup shows 40-65 risk score
  4. Balanced concerns and highlights

🟢 Low Risk Website Tests

These sites should show GREEN "LOW" badges with minimal warnings:

📱 Signal Terms 🍎 Apple Terms 🦊 Mozilla Terms

Expected Results:

  1. Extension badge turns GREEN with "LOW" text
  2. Minimal or no notifications
  3. Popup shows 15-35 risk score
  4. More highlights than concerns

⚪ No Terms Website Tests

These sites should show GRAY "?" badges (no terms detected):

🌐 Example.com 🔧 HTTPBin

Expected Results:

  1. Extension badge shows GRAY "?" text
  2. No notifications appear
  3. Popup shows "No terms found" message
  4. Links section empty or minimal
